Cloud Function: Cloud Function: サーバーレスの計算サービスで、開発者は簡単な単一目的の関数をイベントに基づき自動で実行できます。サーバーインフラの管理は不要です。
Cloud Run: Cloud Run: 完全管理型プラットフォームで、開発者はコンテナ化されたアプリケーションを実行できます。トラフィックに応じて自動でスケーリングし、HTTP駆動のマイクロサービスのデプロイに適しています。
Cloud SQL: Cloud SQL: 完全管理型のリレーショナルデータベースサービスで、MySQL、PostgreSQL、SQL Serverをサポートし、容易に設定、維持、管理が可能です。
