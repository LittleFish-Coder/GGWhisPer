Auf der Google-Plattform nutzen wir verschiedene Cloud-Dienste, um eine vollständige Anwendungsarchitektur zu erstellen. Cloud Run ermöglicht uns die einfache Bereitstellung von containerisierten Anwendungen und die automatische Skalierung basierend auf dem Traffic. Wenn einfache Funktionen ausgeführt werden müssen, bietet Cloud Function eine serverlose Lösung. Für die Datenspeicherung stehen Cloud SQL und Artifact zur Verfügung. Lexus あ.
